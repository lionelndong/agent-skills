# Chapter 8: Value Offer — The Thought Process

## Core Idea
Building a Grand Slam Offer requires **divergent** thinking — generating many solutions to one problem — not the **convergent**, single-answer thinking school trained into us. "Life will pay you for your ability to solve using a divergent thought process."

## Frameworks Introduced
- **Convergent vs. Divergent Thinking**:
  - **Convergent** — many known variables, fixed conditions, one right answer (math; e.g., "how many salespeople must I hire?"). Binary: right or wrong.
  - **Divergent** — multiple variables (known & unknown), dynamic conditions, *multiple* right answers, "and one answer that is way more right than the others."
- **The Brick Exercise** — set a 120-second timer; list as many different uses for a brick as possible. Then reconsider the assumptions (How big? What material? What shape? Holes?) to unlock many more uses (paperweight, doorstop, weight for resistance training, gold brick as payment/retainer of value, plastic brick as floatation, holed brick as pen/plant holder, etc.).

## Key Concepts
- **Offer building blocks** — "Every offer has building blocks, the pieces that when combined make an offer irresistible." Use divergent thinking to combine elements into value.
- Find the customer's desire first, then "devise how many ways I could create value with my 'brick.'"

## Worked Example: The Brick, Re-seen
Most people list ~5 obvious uses under time pressure. Hormozi's expanded list (after questioning size/material/shape) includes: paperweight, doorstop, building material, fishbowl home, plant holder (holed), painted trophy, rustic decor, window-breaker, mini-brick mural, resistance weight, wedge/leveler, pen holder, lego toy, floatation device (plastic), payment (gold), value store (gold), flagpole holder (holed), jumbo-brick seat. The lesson: challenge fixed assumptions to multiply solutions.

## Mental Models
- *Think of the brick's variations* — the same way a single deliverable can be delivered one-on-one, small group, or one-to-many; DIY/DWY/DFY; live or recorded.
- *You get paid for thinking* — this is high-leverage work you do once.

## Anti-patterns
- Defaulting to the first, "logical," conventional solution (convergent reflex) when building an offer.

## Key Takeaways
1. Switch deliberately into divergent mode before designing an offer.
2. Question every assumption about how a solution must be delivered.
3. One offer has many possible building blocks; generate widely before you trim.
